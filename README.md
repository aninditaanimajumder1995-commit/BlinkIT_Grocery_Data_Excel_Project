📊 BlinkIT_Grocery_Data_Excel_Project
👉 Analyzing data beyond numbers — revealing the story behind sales and customers.

🧠 Overview

Analyzed grocery sales data using Microsoft Excel to uncover actionable insights on sales performance, customer behavior, and regional trends. This project demonstrates the ability to transform raw data into meaningful business decisions.

🔧 Key Skills
Excel Functions: XLOOKUP, SUMIF, COUNTIF, IF
Pivot Tables & Charts for data summarization
Data Cleaning & Formatting
Conditional Formatting for insights visualization

📈 Key Insights
Identified top-performing products and regions driving revenue
Observed patterns in repeat customer behavior
Highlighted sales trends and performance gaps
